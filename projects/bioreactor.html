Bioreactor Control System

Agitator Control for Bioreactor Prototype at ITB

📖 Overview

An agitator control system developed for a bioreactor prototype at Lab Sistem Kendali dan Komputer (LSKK), Institut Teknologi Bandung (ITB). The system controls and monitors the mixing speed for optimal performance.

🎯 Problem & Solution

Bioreactors require precise control of mixing speed for optimal growth conditions. Developed and calibrated an agitator control system that precisely controls rotation speed with feedback control and real-time display.

✨ Key Achievements

  • Speed Calibration

    Aligned actual rotation speed with displayed value

  • Control Accuracy

    Improved control accuracy and system performance

  • Mixing Efficiency

    Enhanced through precise speed control

  • Data Analysis

    Comprehensive testing and performance evaluation
