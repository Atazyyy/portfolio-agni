Caku Automation

AI-Enhanced Personal Finance Automation with n8n

View on GitHub

📖 Overview

Caku Automation is an AI-enhanced personal finance automation system that enables receipt processing through a Telegram bot interface. Built with n8n workflow automation, the system processes uploaded receipts using Google Gemini AI for text extraction and stores data in Google Sheets for easy access and analysis.

Hosted via Docker with ngrok for public webhook access, this automation demonstrates end-to-end workflow orchestration skills.

🔄 Workflow Automation

  • Telegram Bot Integration

    Interactive bot for receiving commands and receipt uploads directly from users

  • Receipt Image Processing

    File upload handling with Base64 encoding for OCR processing pipeline

  • AI-Powered OCR

    Google Gemini AI extracts structured data from receipt images automatically

  • Google Sheets Integration

    Automated data storage and synchronization with cloud spreadsheets

  • Error Handling

    Telegram notifications for workflow errors and status updates

  • Manual Input Support

    Text-based expense entry for receipts that cannot be photographed
