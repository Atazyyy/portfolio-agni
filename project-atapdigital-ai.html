AtapDigital AI Prompt Library

Social Media Workflow Automation with Claude API

View on GitHub

📖 Overview

AtapDigital AI Prompt Library adalah web application yang menyediakan 12+ prompt templates untuk workflow social media. Project ini menggunakan Claude API untuk menghasilkan konten berkualitas tinggi.

Dirancang dengan glassmorphism UI yang modern dan interaktif, dilengkapi fitur export ke PDF dan Markdown untuk kemudahan dokumentasi.

✨ Features

  • 12+ Prompt Templates

    Template siap pakai untuk berbagai use case social media workflow

  • Claude API Integration

    Output berkualitas tinggi dengan AI language model Claude

  • Glassmorphism UI

    Modern glass-effect design dengan smooth animations

  • PDF & Markdown Export

    Export hasil prompt ke format PDF atau Markdown

  • Interactive Demo

    Demo page untuk mencoba prompt templates secara langsung
