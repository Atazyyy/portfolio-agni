Sumo Robot

Arduino Robotics Competition
🥊 📷 Tambahkan screenshot project di sini

Overview

A combat sumo robot designed and built for robotics competition. The robot uses distance sensors to detect the opponent and the edge of the ring (dohyo), implementing aggressive strategies to push the opponent out of the arena.

Problem

Sumo robot competitions require robots that can detect opponents, navigate the arena boundaries, and implement offensive strategies - all while being robust enough to withstand physical competition.

Solution

Built a competitive sumo robot with distance sensors for opponent detection and edge detection. The robot is designed to be aggressive and durable, with optimized movement control for competitive performance.

Key Features

  • Distance sensors for opponent detection
  • Edge detection to stay within the dohyo
  • Aggressive push strategy
  • Robust chassis design for competition
  • Optimized motor control

Hardware Components

  • Arduino microcontroller
  • Distance sensors (ultrasonic/IR)
  • High-torque DC motors
  • Motor driver
  • Durable chassis and bumpers

Competition Strategy

The robot uses a state-machine control strategy. When an opponent is detected, it charges forward aggressively. When near the edge, it backs away to avoid being pushed out. The strategy was optimized through testing and iteration.

Technologies Used

  • Hardware: Arduino, Distance Sensors, DC Motors
  • Programming: C++ (Arduino IDE)
  • Strategy: State Machine Control
